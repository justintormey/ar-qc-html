Order of operations
  1. Set a Room PIN above. Both devices use the same PIN to pair over WebRTC.
  2. On the headset: open the Start Demo card. The HUD loads with the Welcome panel.
  3. On a second device: open the Controller card. Wait for the status to show connected.
  4. The participant picks up Part A → operator taps PART A. The headset plays the PASS verdict overlay.
  5. Repeat for Part B (REWORK + 3D ghost) and Part C (SCRAP). Then the Completion summary.
